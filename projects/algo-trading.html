The Universal Plug for LLMs
What MCPs are and should we pay attention? (Source: Krishnam Gupta - TrueFoundry)
What is an MCP?
A standard language for AI agents to talk to financial tools
MCPs let any AI model — LLM or agent — plug into brokers, data feeds, and trading tools without custom code for each integration. One protocol, any tool.
"Like USB-C for AI — instead of a different cable for every device, one standard connects everything."
— Anthropic, inventors of MCP
Standardised — one API surface, not N integrations
Composable — mix broker + data + news MCPs freely
Agent-native — built for LLMs acting autonomously
